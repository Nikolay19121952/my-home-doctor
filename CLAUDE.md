# Мой домашний доктор

PWA-приложение для ведения медицинской карты семьи, дневника здоровья и хранения результатов анализов.

## Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков)
- PWA (Progressive Web App) — работает офлайн, устанавливается на телефон
- localStorage для данных профилей, IndexedDB для файлов (будущее)

## Структура

- `index.html` — единственная страница (SPA, навигация через JS)
- `css/styles.css` — все стили, CSS-переменные для темы
- `js/app.js` — навигация и инициализация
- `js/storage.js` — CRUD-обёртка над localStorage
- `js/profiles.js` — раздел «Семья» (профили членов семьи)
- `js/ui.js` — общие UI-функции (тосты, модалки)
- `sw.js` — Service Worker для офлайн-режима
- `manifest.json` — PWA-манифест

## Запуск

Для работы PWA и Service Worker нужен HTTP-сервер:

```bash
python -m http.server 8080
```

Затем открыть http://localhost:8080

## Хранение данных

- Ключ `mdd_profiles` — JSON-массив профилей членов семьи
- Все данные хранятся локально на устройстве пользователя

## Дизайн

- Целевая аудитория: пенсионеры
- Крупный шрифт (18-20px), большие кнопки (48px+)
- Высокий контраст, без скрытых меню и сложных жестов
- При разработке в DevTools > Application > Service Workers включить "Update on reload"
