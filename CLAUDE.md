# Мой домашний доктор

PWA-приложение для ведения медицинской карты семьи, дневника здоровья и хранения результатов анализов.

## Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков)
- PWA (Progressive Web App) — работает офлайн, устанавливается на телефон
- localStorage для текстовых данных, IndexedDB для изображений анализов

## Структура

- `index.html` — единственная страница (SPA, навигация через JS)
- `css/styles.css` — все стили, CSS-переменные для темы
- `js/app.js` — навигация, инициализация модулей, сводка на главной
- `js/ui.js` — общие UI-функции (тосты, модалки, выбор профиля, цветовая индикация)
- `js/db.js` — обёртка над IndexedDB для хранения изображений
- `js/storage.js` — CRUD-обёртка над localStorage для всех типов данных
- `js/profiles.js` — раздел «Семья» (профили членов семьи)
- `js/diary.js` — раздел «Дневник здоровья» (показатели, SVG-графики)
- `js/analyses.js` — раздел «Анализы» (метаданные + фото из IndexedDB)
- `js/reminders.js` — раздел «Напоминания» (уведомления, повтор)
- `sw.js` — Service Worker для офлайн-режима
- `manifest.json` — PWA-манифест

## Запуск

Для работы PWA и Service Worker нужен HTTP-сервер:

```bash
npx http-server . -p 8080 -c-1
```

Затем открыть http://localhost:8080

## Хранение данных

- `mdd_profiles` — JSON-массив профилей членов семьи
- `mdd_diary` — JSON-массив записей дневника здоровья
- `mdd_analyses` — JSON-массив метаданных анализов
- `mdd_reminders` — JSON-массив напоминаний
- IndexedDB `mdd_images` — хранилище изображений анализов (blob)
- Все данные хранятся локально на устройстве пользователя
- При удалении профиля каскадно удаляются все связанные данные

## Дизайн

- Целевая аудитория: пенсионеры
- Крупный шрифт (18-20px), большие кнопки (48px+)
- Высокий контраст, без скрытых меню и сложных жестов
- Цветовая индикация показателей: зелёный (норма), оранжевый (повышено), красный (высокое)
- При разработке в DevTools > Application > Service Workers включить "Update on reload"
